Welcome to Aurora!

What is Aurora?

Aurora is a private discord bot for the discord server Aurora!

It helps us at Aurora manage our server and customize it in more advanced ways than would normally be possible.

Can I get Aurora myself?

Due to the bot being a private bot for our server, it is specialized for our server and won't work on other servers.

However, the bot is partly open source, so you can take a look at the source code and take inspiration for your own discord bot.

Bot Command Documentation

Basic bot commands

Command Description
Prefix
!level {user} (opt) /level {user} (opt) Shows the level of the user. If no user is set, it will show your own level
!leaderboard {amount} /leaderboard {amount} Bans a user

Moderation bot commands

Command Description Required permissions
Prefix
!ban {user} {reason}(opt) /ban {user} {reason}(opt) Bans a member Ban members
!unban {targetId} /unban {targetId} unbans a member Ban members
!kick {user} {reason}(opt) /kick {user} {reason}(opt) Kicks a member Kick members
!timeout {user} {duration} {reason}(opt) /timeout {user} {duration} {reason}(opt) Times a member out for a given amount of time Moderate members
!purge {amount} /purge {amount} Deletes a given amount of messages from the current channel Manage messages
