Privacy Policy

Introduction

Welcome to Aurora! This Privacy Policy explains how we collect, use, and protect your personal data when you interact with our bot.

Data Collection

We may collect the following data:

  • - Discord username and user ID
  • - Messages and commands sent to the bot
  • - Any other data you provide during interactions with the bot

Data Usage

We use your data to:

  • - Provide and improve the functionality of the bot
  • - Maintain and troubleshoot the bot
  • - Respond to inquiries and support requests

Data Storage

Your data is only stored as long as its needed for the bots functionality. E.g. if the user sends a command to the bot, the user data is only stored as long as its needed for the command to properly finish. After finishing the command, all associated data will no longer be used or stored Other than that we only use data that is provided by discord directly.

Third-Party Services

None of your data is ever shared with any third-party services.

Changes to the Privacy Policy

We may update this Privacy Policy from time to time. The latest version will be available here. This policy was last updated on March 29th, 2025.
